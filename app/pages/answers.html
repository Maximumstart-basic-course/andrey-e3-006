Что такое EcmaScript?

ECMAScript - встраивамый имеющий расширение и не имеющий средств ввода-вывода язык програмирования.

Для чего используются и чем отличаются атрибуты async и defer?

Атрибуты async и defer позволяют при постепенной загруке html-страницы выполнять скрипты. Атрибут defer позволяет скриптам выполнится последовательно, async разрешает скриптам выполняться помере загрузки. Скрипт defer срабатывает, когда весь документ html обработан браузером; скрипт async начинает выполнятся одновременно с загрузкой страницы html.

Почему предпочтительнее подключать скрипты перед </body>, а не в <head>...</head>?

Подклчение скриптов оптимально перед закрівающим тегом </body> оптимизирует скорость загрузки данных на страницу.

Какие основные отличия интерпретируемого языка программирования от компилируемого?

Компилирование программы предусматривает перевод ее в бинарную систему и ее выполнение начинается только после полного завершения этого процесса. Интерпретированый язык программирования предусматривает его считывание программой машины-исполнителя с параллельным его выполнением.

Что такое языки высокого и низкого уровня? Какие ты знаешь?

Низкоуровневые языки программирования наиболее близки синтаксису и симантике языку машины-исполнителя, высокоуровневые - более компактны и удобны, но нуждаются в более длительном компилировании.

Для чего используются переменные? Какие ключевые слова используются для создания переменных? Какие есть отличия в переопределении значений при задании переменной через let, const и var?

Переменные используются для хранения информации. Для создания переменной используются ключевые слова var, let и const. Переменная, объявленная через var видна везде в функции и может существовать до объявления (равна undefined). Переменная, объявленная через let, видна только в рамках блока, в котором объявлена и до объявления не существует.Переменная через var задается раз для всех повторений цикла, переменная через let - каждый раз по-новому задается при каждом новом повторении цикла.Объявление через const создает переменную, которая не может изменяться. В остальном const похожа на let.

Может ли имя переменной начинаться с цифры, символов _ или $?

Первый символ в имени переменной не может начинаться с цифры.

Какие слова запрещено использовать в названии переменной?

var, class, return, export.

Что такое camelCase и snake_case?

Стиль CamelCase предусматривает написание слов внутри фразы слитно, причем каждое слово начинается с прописной буквы. Snake case - cтиль написания фраз, составные слова которых разделяются регистром подчеркивания и не имеют пробелов; каждое слово пишется с маленькой буквы.

Что такое "типы данных"? Какой оператор позволяет определить тип данных?

Типы данных - это множество значений и операций на этих значениях. Оператор typeof возвращает строку, которая описывает тип данных переменной.

Что такое "динамическое приведение типов данных"? Можно ли писать JavaScript со статическим приведением типов?

Динамическое приведение типов данных связано с их определением во время выполнения программы. Статичские типы данных могут быть использованы в JavaScript с помощью инструментов Flow и TypeScript.

Чем отличаются операторы "==" и "==="?

== оператор равенства; === - оператор строго равенства;

Какие преимущества дают обратные кавычки (`) при указании строки?

В строчках одинарными обратными ковычками возможен перевод строки, вставка выражений в виде ${...}.
